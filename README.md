# APEX-
Apex Legends 战绩查询小程序 基于微信云开发 + Mozambiquehe.re API 的 Apex Legends 战绩查询工具。用户输入游戏 ID 即可实时查询段位、KD、场均伤害、胜率等核心数据，支持 Steam / Origin / PS / Xbox / Switch 全平台玩家搜索。 核心功能 玩家搜索 → 综合战绩（KD、伤害、胜率）→ 段位展示（RP + 图标）→ 传奇数据 → 最近搜索快捷重查 技术栈 微信原生小程序 + 云函数（Node.js）+ Mozambiquehe.re API 架构 小程序前端 → 云函数（API 代理）→ Mozambiquehe.re → 返回数据 → 渲染战绩页
